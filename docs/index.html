关于我们

浙江大学国旗仪仗队成立于1997年，是一支由在校大学生组成的国旗护卫队伍。我们承担着学校日常升旗任务以及重大活动的仪仗任务。

多年来，国旗仪仗队始终秉承"爱国、荣校、奉献、进取"的精神，以严谨的作风、规范的动作、昂扬的精神风貌，展现了浙大学子的风采。

本平台旨在记录和展示国旗仪仗队的发展历程，保存珍贵的影像资料，传承仪仗精神。
